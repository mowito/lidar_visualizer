Instructions:


1: Create obstacles, by clicking on the canvas.

2: Select LiDAR from the drop-down menu above.

3: Hit "Play" and move the cursor on the canvas.

4: To add more obstacles, hit "Stop" and add obstacles.

5: To remove obstacles, hit "Reset".

6: Modify the values at the top, to test for custom values.

Note: In case of range of values, the max value is used for visualization

Note: The tool is currently designed for 2D Visualization only

If you want a new LiDAR to be added, please contact us!
